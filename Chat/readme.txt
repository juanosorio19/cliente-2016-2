

el server se ejecuta:  ./server

el cliente se ejecuta: ./client <dirección_ip:puerto>   //localhost:4242

para registrarse: register <username> <password>

para iniciar sesión: login <username> <password>

para mandar un mensaje de texto: msg <destino> <mensaje>

para registrarse en un grupo: gregister <grupo> <usuario>

para mandar un mensaje de texto a un grupo: gmsg <grupo> <mensaje>

para mandar una nota de voz: voice <destinatario>

para hacer una llamada: call <destinatario>
